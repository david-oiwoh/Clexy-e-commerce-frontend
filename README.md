# Clexy Ecommerce Frontend

Clexy is a modern and responsive ecommerce frontend interface built using HTML and CSS. This project showcases a premium electronics store UI with advanced layout structure and interactive design elements.

---

## 🚀 Features

- Fully responsive ecommerce layout  
- Product listing grid with hover effects  
- Product detail page UI  
- Add-to-cart interaction UI  
- Modern UI design with smooth animations  
- Clean and structured layout  

---

## 🛠️ Technologies Used

- HTML5  
- CSS3  
- Font Awesome
